Python 直接赋值、浅拷贝和深度拷贝解析

在 Python 中，赋值、浅拷贝和深拷贝是常见的操作，它们之间的主要区别在于它们如何处理对象中的引用关系。下面逐一解析它们的不同：

1. 直接赋值（赋值操作 =）

直接赋值是将一个对象的引用（内存地址）赋给另一个变量，而不是复制对象的内容。这意味着两个变量指向相同的对象。

  • 行为：赋值后的两个变量指向同一个对象，修改一个变量的内容会影响另一个变量。

2. 浅拷贝（Shallow Copy）

浅拷贝创建一个新的对象，但该对象内的元素仍然是原对象中元素的引用。如果原对象是一个复合对象（如列表或字典），那么浅拷贝的对象和原对象会共享内部的子对象。

  • 行为：对外部对象的修改不会影响原对象，但对内部子对象的修改会影响原对象。

  • 常见方法：

    • 使用切片操作（对于列表等可变序列）：b = a[:]。

深度拷贝创建一个新的对象，并递归地复制对象及其所有嵌套对象。这样，原对象和拷贝对象之间完全独立，修改任何一个对象都不会影响另一个。

总结：

  • 直接赋值：两个变量指向同一个对象，修改一个变量会影响另一个。
  • 浅拷贝：创建一个新的对象，但内部的嵌套对象仍然是原对象的引用，修改嵌套对象会影响原对象。
  • 深度拷贝：创建一个新的对象，并递归地复制所有嵌套对象，修改任何一个对象都不会影响另一个。

生成器

生成器的本质就是迭代器。在每次迭代时返回一个值，直到抛出 StopIteration 异常。

创建生成器的方式：

（1）生成器函数

使用关键字 yield，函数中出现了 yield，那该函数就是一个生成器函数；生成器函数执行的时候，并不会立即执行函数，得到的是生成器。

  • 生成器可以通过 yield 返回数据
  • 可以分段执行函数的内容，通过 next，执行到下一个 yield 的位置

和 return 相比，yield 除了可以返回相应的值，还有一个更重要的功能，即每当程序执行完该语句时，程序就会暂停执行。

不仅如此，即便调用生成器函数，Python 解释器也不会执行函数中的代码，它只会返回一个生成器（对象）

闭包

内部函数引用外部函数的变量（非全局变量），外部函数返回内部函数（返回内部函数的函数名），内部函数就是闭包函数。

所以闭包有以下特点：

  • 函数嵌套
  • 内部函数使用外部函数的变量

列出python中可变数据类型和不可变数据类型，简述原理

在 Python 中，数据类型根据其是否可以被修改分为 可变数据类型（Mutable） 和 不可变数据类型（Immutable）。它们的主要区别在于对象的修改是否会影响原始数据及内存管理的方式。

1. 可变数据类型（Mutable Types）

可变数据类型是指对象的内容可以在创建后被修改。对这些类型的对象进行修改时，Python 会在原址修改对象，而不会创建新的对象。

常见的可变数据类型：

  • 列表（list）：可以修改列表中的元素，改变列表的大小（添加、删除元素）。
  • 字典（dict）：可以修改字典的键值对。
  • 集合（set）：可以修改集合的元素（添加、删除）。
  • 自定义对象：如果类中包含可变的数据结构（如列表、字典等），对象的属性可以修改。

原理：

  • 内存管理：可变数据类型的对象存储在堆内存中。当对象内容发生变化时，内存地址不变，直接修改其内容。
  • 修改行为：修改可变对象时，原对象的内存地址保持不变，不会创建新对象。直接修改对象内容的同时，其他引用该对象的变量会看到变化。

2. 不可变数据类型（Immutable Types）

不可变数据类型是指对象的内容一旦创建就不能被修改。对这些类型的对象进行修改时，实际上会创建一个新的对象。

常见的不可变数据类型：

  • 整数（int）：整数的值一旦赋值后无法更改。
  • 浮点数（float）：浮点数一旦赋值后无法更改。
  • 字符串（str）：字符串一旦创建后，无法修改其中的字符。
  • 元组（tuple）：元组中的元素无法修改，但可以创建一个新的元组来替代。
  • 冻结集合（frozenset）：冻结集合是不可变的集合，不能修改其内容。
  • 字面量常量：如布尔值 True 和 False，以及 None。

原理：

  • 内存管理：不可变数据类型的对象存储在内存中时，其内容一旦创建就无法改变。如果要修改内容，Python 会创建一个新的对象，并且将原对象标记为可回收。
  • 修改行为：不可变对象一旦创建，其内容不能被改变。如果尝试改变，实际上是生成了一个新的对象，并分配给一个新的内存地址。

3. 可变与不可变的内存管理差异

  • 可变对象：内存中存储的是对象的引用（地址）。修改对象时，不会重新分配新的内存地址，直接修改原来的内容。因此，所有引用该对象的变量都会看到变化。
  • 不可变对象：当不可变对象被修改时，Python 会创建一个新的对象，并将新的对象分配一个新的内存地址，原来的对象会被保留，而与其关联的变量会指向新的对象。由于其不可变性，其他变量无法看到对原对象的修改。

什么是Python中的类型转换？

类型转换（Type Conversion） 是将一个数据类型的值转换为另一数据类型的过程。在 Python 中，类型转换分为 显式转换（即手动转换）和 隐式转换（即自动转换）。在某些情况下，Python 会自动进行类型转换，但也可以通过显式的方式进行转换。

转换函数 功能说明 示例
int() 将数据转换为整数。 int("123") → 123
float() 将数据转换为浮点数。 float("3.14") → 3.14
str() 将数据转换为字符串。 str(123) → "123"
list() 将可迭代对象转换为列表。 list((1, 2, 3)) → [1, 2, 3]
tuple() 将可迭代对象转换为元组。 tuple([1, 2, 3]) → (1, 2, 3)
set() 将可迭代对象转换为集合，去除重复元素。 set([1, 2, 2, 3]) → {1, 2, 3}
bool() 将数据转换为布尔类型。空值、零值等转换为 False，其余为 True。 bool(0) → False bool("hello") → True
dict() 将包含键值对的可迭代对象转换为字典。 dict([("a", 1), ("b", 2)]) → {'a': 1, 'b': 2}
chr() 将整数转换为对应的 Unicode 字符。 chr(97) → 'a'
ord() 将字符转换为对应的 Unicode 整数。 ord('a') → 97
bin() 将整数转换为二进制字符串。 bin(5) → '0b101'
hex() 将整数转换为十六进制字符串。 hex(255) → '0xff'
oct() 将整数转换为八进制字符串。 oct(8) → '0o10'

*类型转换的注意事项：

  • 从字符串转换为数字： 当字符串中包含无法转换为数字的字符时，会抛出 ValueError。

  • 浮点数到整数的转换： float() 转换为整数时会丢失小数部分。

  • 布尔类型转换： 例如，0、空字符串、空列表、None 都转换为 False，其余转换为 True。
