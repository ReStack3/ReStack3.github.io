Android类加载流程

在Dex加载流程 - The Peak Tower这篇文章中，详细探讨了dex的加载流程，而类加载的整个过程如下：

步骤 描述 相关技术/组件
1. 编译与打包 将 .java 源代码编译成 .class 文件，然后使用 d8 或 dx 工具将其转换为 .dex 文件，并打包到 APK 中。 d8、dx、APK
2. 应用启动 当应用启动时，系统加载 .dex 文件，准备执行其中的代码。 Dalvik（Android 5.0 之前）、ART（Android 5.0 及以上）
3. 类加载器加载 使用 ClassLoader 加载 .dex 文件中的类。根据不同场景使用不同的类加载器。 DexClassLoader、PathClassLoader、InMemoryDexClassLoader（Android 8.0 及以上）
4. 加载与执行 将 .dex 文件中的类加载到内存中，转换为 Java 类对象并进行执行。 defineClass()（加载类）
5. ART 的 AOT 编译 在 ART 中，.dex 文件可以通过 AOT 编译预先生成机器码，提高启动性能。 AOT（Ahead-of-Time Compilation）
6. 动态加载支持 支持动态加载额外的 .dex 文件，例如插件化和热更新场景。 DexClassLoader（动态加载）

自然的这章，我们将要讨论的是android中如何将类从.dex文件中加载到内存中，转换为 Java 类对象并进行执行。

在一个类加载之前，其类对应的dex文件已经在内存中。

ClassLoader类是所有类加载器的虚基类，在静态/动态加载一个类时会首先调用此虚基类的LoadClass函数。

注意：c = parent.loadClass(name, false);涉及到了双亲委派机制【双亲委派机制是 Java 类加载器（ClassLoader）的核心机制之一，旨在确保 Java 类的加载遵循一种优先级的委派模式。其基本思想是：当前类加载器请求加载一个类时，首先会将请求委派给父类加载器，如果父类加载器无法处理，则由当前类加载器处理。这种机制有助于避免类的重复加载和冲突。】具体可参考之前文章初识JVM - The Peak Tower

findClass

因为BaseDexClassLoader继承了ClassLoader虚类，其他加载器的类基本都继承于BaseDexClassLoader类，BaseDexClassLoader类中重写了findClass方法，所以上一步会去调用BaseDexClassLoader类中的findClass方法。

从类路径中查找类，如果找不到，抛出一个带有附加信息的 ClassNotFoundException 异常，并将其他抑制的异常添加到这个异常中。

pathList.findClass 是查找类的关键步骤。pathList 代表一个类路径，它包含了可能包含类的文件（如 .dex 文件）。

findClass 方法会返回一个 Class 对象（如果找到类），或者返回 null（如果没有找到类）。

BaseDexClassLoader的findclass会继续调用DexPathList的findClass方法

这个 findClass 方法的作用是在 dexElements 中查找类 name。它遍历所有的 Element（每个 Element 通常表示一个 DEX 文件或类加载单元），尝试从中找到指定的类。如果找到，则返回该类；如果没有找到，并且在查找过程中发生了异常，它会将这些异常添加到 suppressed 列表中，然后返回 null。

Element类的findClass方法去调用loadClassBinaryName函数，

该方法的作用是尝试通过 dexFile 加载指定的类。如果 dexFile 不为 null，则通过 dexFile 来加载类；否则，返回 null。

loadClassBinaryName

loadClassBinaryName 是一个公开的方法，它负责加载指定名称的类。

  • name：要加载的类的名称（例如 com.example.MyClass）。
  • loader：用于加载该类的类加载器（ClassLoader）。
  • suppressed：一个 List<Throwable> 类型的参数，用来存储抑制的异常。

该方法通过 defineClass 方法加载类并返回 Class 对象。defineClass 是一个静态方法，它的主要任务是通过 defineClassNative（本地方法）定义类。

defineClassNative

loadClassBinaryName函数接着会调用defineClass->defineClassNative。defineClassNative函数是一个native函数在art虚拟机中实现。
